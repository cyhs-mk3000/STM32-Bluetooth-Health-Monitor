# 基于STM32F103C8T6的人体健康监测系统

## 项目核心亮点 

- ✅ **完整落地**：基于HC-05蓝牙实现健康数据采集→本地显示→手机APP传输→超限报警全链路；

- ✅ **高精度采集**：集成MQ-R-12（心率/血氧/血压）+ DS18B20（体温），采集精度达设计指标；

- ✅ **工程化设计**：硬件做电源/信号完整性优化，软件模块化开发，代码可直接编译运行；

## 核心功能 

| 功能模块         | 实现效果                                                                 | 
|------------------|-------| 
| 多参数采集       | 实时采集血氧(±1%)、体温(±1℃)、心率(±1BPM)、血压(±1mmHg)，支持连续采集； | 
| 本地交互 | OLED 12864显示参数+系统时间，3个按键自定义预警阈值（如心率≥100BPM报警）；| 
| 蓝牙传输 | HC-05蓝牙模块实现数据透传，手机APP（BlueSPP）实时接收/存储数据； |
| 异常报警 | 参数超阈值触发声光报警（蜂鸣器三短声+LED常亮），恢复正常后自动清除；     | 

## 硬件选型（核心清单） 

| 模块类型         | 型号/规格               | 关键设计要点                     |
|------------------|------------------------|----------------------------------| 
| 主控芯片 | STM32F103C8T6 | 72MHz主频，64KB Flash，满足实时性 |
| 多参数传感器 | MQ-R-12 | 单UART接口传输心率/血氧/血压，简化硬件 | 
| 温度传感器 | DS18B20 | 单总线协议，抗干扰能力强    | 
| 无线通信 | HC-05蓝牙模块 | 10米传输距离，串口透传适配手机APP | 
| 显示模块         | OLED 12864（I2C）      | 低功耗，高对比度，实时显示参数   | 
| 电源管理         | RT9193（5V→3.3V）      | 低压差稳压器，四电容滤波降噪声   | 

## 软件核心实现 

### 开发环境 
- 编译器：Keil5 MDK 
- 编程语言：C语言 
- 调试工具：串口助手、万用表、蓝牙调试APP（BlueSPP） 
### 核心代码逻辑（关键片段） 
```c 
// 主函数核心流程（完整代码见 03_Software/Core_Code_For_Interview/main.c） 
int main(void) {  
  System_Init();       // 系统初始（GPIO/USART/I2C/OLED） 
  while(1) {
  KEY_Process();     // 按键消抖与阈值设置      
  Data_Collect();    // 传感器数据采集+数字滤波    
  OLED_Display();    // 本地参数+时间显示    
  Alarm_Detect();    // 异常检测与声光报警    
  Bluetooth_Send();  // 蓝牙数据透传（200ms/次）  
  } 
} 
// 蓝牙数据封装格式（保证APP解析稳定） 
void Bluetooth_Send(void) {  
  sprintf(buf,"HR:%d,SPO2:%d,TMP:%.1f,HBP:%d,LBP:%d\r\n",         heart_rate, spo2, temperature, high_bp,low_bp); 
  USART1_SendString(buf); // 发送至HC-05模块 
}
```
