# STM32-Bluetooth-Health-Monitor 

基于STM32F103C8T6的本地人体健康监测系统，通过HC-05蓝牙实现多参数采集数据的本地传输与异常报警，无远程/云平台依赖。 

##  核心特性 

- 多传感器数据采集：支持温度（DS18B20）、心率、姿态（MPU6050）、温湿度（DHT11）； 

- 蓝牙本地通信：HC-05模块与手机BlueSPP APP实时传输数据；

- 异常报警：心率/温度超阈值时，蜂鸣器+LED声光报警； 

- 本地数据显示：0.96寸OLED屏实时展示采集数据与系统状态。 

## 技术栈
- 主控：STM32F103C8T6 
- 通信：HC-05蓝牙模块（UART） 
- 开发环境：Keil5 
- 外设：OLED（I2C）、蜂鸣器、LED、多类型传感器
